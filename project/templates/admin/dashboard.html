There are many things that you can do in this system, they are listed below:


Lookup a Book

You can search for different books in the library. Different filters are provided and you can also search for books that are currently borrowed. Just go to "View Library Books" and fill in any detail you know or just view all the books. Do note that choosing to view all the books in the Library may load longer.


Approve or Deny Books Requests

You can start accepting or denying book requests from students by going to "Book Approvals". Keep in mind the different criteria that a student must pass in order to borrow or return a book:
  • Students can only borrow a maximum of ten (10) books. Deny other requests if they go over.
  • The student should present the book in-front of you, together with their ID, which you will check to verify their identity before approval.
  • Students that are returning books must present the book to the library before the return is processed.
  • NOTE: Students who fail to return a book by the end of the school year will not be shown their grade until they return the book / provide a replacement / pay for the book.


Process Book Returns

Just go to the "Book Returns" page to process the books being returned by students. Make sure that you only process the returns IF the book is actually returned by the student.


Upload Books

If someone donates or gives a new book to the library, you should add the book to the database. To do that, just go to "Upload Books" and upload a CSV file containing the proper requirements listed on the page.


Monitor Inventory

You can go to "Lost Books" to see a list of all the books that have not been returned by students.
